# FEATURES: FuelKeeper 기능 명세서

> 각 기능은 **ID**, **우선순위**, **설명**, **수용 기준(AC)**, **엣지 케이스**로 구성된다.  
> Cursor에게 특정 기능 구현을 맡길 때는 해당 섹션을 통째로 참조시켜라.

---

## 우선순위 정의

- **P0**: v1 출시 필수 (없으면 제품이 아님)
- **P1**: v1 출시 권장 (있어야 경쟁력 확보)
- **P2**: v1.x에서 추가 (포폴 데모에 포함되면 좋음)

---

## F-01: 주변 주유소 지도 탐색 [P0]

### 설명
사용자의 현재 위치를 기준으로 반경 3km 이내의 주유소를 지도에 마커로 표시한다. 마커에는 가격이 노출되며, 최저가는 강조 색상으로 표시된다.

### 수용 기준 (Acceptance Criteria)
- [ ] 앱 첫 실행 시 위치 권한을 요청한다.
- [ ] 권한 허용 시 현재 위치를 중심으로 지도가 로드된다.
- [ ] 반경 3km 내 휘발유 판매 주유소가 마커로 표시된다.
- [ ] 마커에는 가격(원 단위)이 함께 표시된다.
- [ ] 최저가 마커는 **다른 색(녹색)** 및 **약간 큰 크기**로 강조된다.
- [ ] 정부 최고가 초과 주유소는 **빨간색** 마커로 표시된다.
- [ ] 마커를 탭하면 하단에 주유소 상세 정보 바텀시트가 열린다.

### 엣지 케이스
- 위치 권한 거부 시 → 수동 지역 선택 화면으로 이동 (F-09 참조)
- 반경 내 주유소가 5개 미만일 경우 → 자동으로 반경 5km로 확장
- GPS 신호 불량 시 → 3초 후 "위치를 가져올 수 없습니다" 오류 + 수동 선택 제공

### 기술 노트
- `geolocator` 패키지로 위치 조회
- `google_maps_flutter` 또는 `flutter_naver_map`로 지도 렌더링 (네이버 지도가 한국 도로/주유소 데이터 정확도 우위)
- 마커 클러스터링 필요 (`flutter_map_marker_cluster` 또는 커스텀 구현)

## F-02: 주유소 리스트 뷰 [P0]

### 설명
지도 하단 탭으로 리스트 뷰 전환 가능. 최저가 순으로 정렬된 주유소 목록을 표시한다.

### 수용 기준
- [ ] 지도/리스트 토글 버튼으로 뷰 전환 가능
- [ ] 각 카드는 **주유소명, 브랜드 로고, 가격, 거리(m/km), 정부 최고가 대비 금액**을 포함한다.
- [ ] 기본 정렬은 "가격 낮은 순", 토글로 "거리 가까운 순" 전환 가능
- [ ] 브랜드 필터 칩(All / SK / GS / 현대 / S-Oil / 알뜰) 제공
- [ ] 유종 필터(휘발유/경유/고급휘발유) 제공, 기본은 휘발유
- [ ] 무한 스크롤 (한 번에 20개씩 로드)

### 엣지 케이스
- 필터 결과 0건 시 → "조건에 맞는 주유소가 없습니다" 빈 상태 UI
- 네트워크 오류 시 → 마지막 캐시 + "30분 전 정보" 뱃지 표시

## F-03: 주유소 상세 정보 [P0]

### 설명
주유소 카드 탭 또는 마커 탭 시 상세 정보 화면 표시.

### 수용 기준
- [ ] 주유소명, 주소, 브랜드, 셀프 여부, 세차장/편의점/정비소 부가 서비스 표시
- [ ] 유종별 현재 가격 표시 (휘발유/경유/고급휘발유/등유)
- [ ] 정부 최고가 대비 차이 명시 ("정부 최고가 대비 -62원")
- [ ] 최근 7일 가격 변동 차트 (**P1**)
- [ ] 버튼 3개: [전화걸기] [길찾기] [즐겨찾기 추가]
- [ ] 전화걸기 → `url_launcher`로 `tel:` 스킴 호출
- [ ] 길찾기 → 네이버 지도/T맵 앱 스킴 호출 (설치 여부 분기)

### 엣지 케이스
- 전화번호 없는 주유소 → [전화걸기] 버튼 비활성화
- 모든 길찾기 앱 미설치 시 → Google Maps 웹으로 폴백

## F-04: 관심 주유소 (즐겨찾기) [P0]

### 설명
자주 가는 주유소를 즐겨찾기로 등록하고, 별도 탭에서 관리한다.

### 수용 기준
- [ ] 상세 화면 또는 리스트 카드에서 별(⭐) 아이콘으로 즐겨찾기 토글
- [ ] 즐겨찾기 탭에서 등록한 주유소를 가격순/등록순 정렬 가능
- [ ] 즐겨찾기 주유소의 현재 가격은 24시간 이내 데이터로 표시
- [ ] 최대 10개까지 등록 가능 (초과 시 토스트 안내)

### 엣지 케이스
- 오프라인에서 즐겨찾기 조회 → 마지막 캐시 데이터 + "오프라인" 뱃지

---

## F-05: 가격 변동 알림 [P1]

### 설명
즐겨찾기 주유소의 가격이 변동되면 로컬 푸시 알림으로 통지한다.

### 수용 기준
- [ ] 앱이 포그라운드에 들어올 때마다 즐겨찾기 주유소 가격을 재조회
- [ ] 직전 저장 가격과 차이가 ±20원 이상이면 로컬 알림 발송
- [ ] 알림 본문: "[주유소명] 휘발유 -30원 ↘️ 1,956원"
- [ ] 설정 화면에서 알림 on/off 가능
- [ ] 알림 탭 시 해당 주유소 상세 화면으로 딥링크

### 엣지 케이스
- 앱이 백그라운드에 오래 있다가 복귀 → 최대 5건까지만 알림 묶음 표시
- 알림 권한 거부 → 설정 화면에서 앱 설정으로 이동하는 버튼 제공

### 기술 노트
- `flutter_local_notifications` 패키지
- v1은 **포그라운드 진입 시 체크** 방식으로 구현 (백그라운드 태스크는 v1.1에서 WorkManager로 확장)

## F-06: 주유 기록 (가계부) [P0]

### 설명
주유 후 금액을 입력하면 자동으로 리터/단가/연비를 계산해 기록한다.

### 수용 기준
- [ ] [+ 주유 기록] FAB으로 기록 추가 모달 오픈
- [ ] 입력 필드: **결제 금액(원)**, **주유소 선택**, **현재 주행거리(km, 선택)**, **유종**
- [ ] 주유소 선택 시 해당 주유소의 현재 가격이 자동 입력 (수정 가능)
- [ ] 리터 수 = 결제 금액 / 유가 (자동 계산)
- [ ] 주행거리 입력 시 직전 기록과 비교해 **연비(km/L)** 자동 계산
- [ ] 기록 리스트는 날짜별 그룹핑
- [ ] 기록 삭제/수정 가능 (스와이프 제스처)

### 엣지 케이스
- 첫 기록인데 주행거리 입력 → 연비 계산 불가 안내 ("다음 주유 기록부터 연비 계산됩니다")
- 주행거리가 이전보다 작음 → 경고 + 저장 여부 재확인
- 주유소 선택 안 하고 수동 입력 → 가능하되 "직접 입력" 태그

## F-07: 월별 주유 통계 [P1]

### 설명
주유 기록을 기반으로 월별 지출 및 추이를 시각화한다.

### 수용 기준
- [ ] 통계 탭에서 이번 달 총 지출, 총 주유량(L), 평균 단가 표시
- [ ] 최근 6개월 월별 지출 Bar Chart (`fl_chart` 사용)
- [ ] 평균 단가 추이 Line Chart
- [ ] 평균 연비 (km/L) 카드 표시
- [ ] 월 전환 드롭다운 (1월, 2월, ...)

### 엣지 케이스
- 기록 0건 → "주유 기록을 추가하면 통계를 볼 수 있어요" 빈 상태 UI

## F-08: 홈 화면 위젯 [P1]

### 설명
앱을 실행하지 않고 홈 화면에서 최저가 주유소 3곳을 확인한다. **Android 우선**, iOS는 v2.

### 수용 기준
- [ ] 위젯 크기: 2x2 (작은 버전), 4x2 (큰 버전)
- [ ] 작은 버전: 최저가 1곳만 표시 (브랜드, 가격, 거리)
- [ ] 큰 버전: TOP 3 리스트
- [ ] 위젯은 30분마다 자동 갱신
- [ ] 위젯 탭 시 해당 주유소 상세 화면으로 앱 실행 + 딥링크

### 기술 노트
- `home_widget` Flutter 패키지 사용
- Android: Kotlin으로 `AppWidgetProvider` 구현 필요
- **면접 어필 포인트:** Flutter ↔ 네이티브 브릿지 구현 경험

### 엣지 케이스
- 위젯 추가 시 위치 권한 없음 → 위젯에 "앱을 열어 위치 권한을 허용해주세요" 메시지 표시

## F-09: 수동 지역 선택 [P0]

### 설명
위치 권한이 없거나 GPS 오류 시 사용자가 직접 시/군/구를 선택해 해당 지역의 주유소를 조회한다.

### 수용 기준
- [ ] 2-Depth 드롭다운: 시도 → 시군구
- [ ] 선택한 지역의 주유소 평균 가격을 상단에 표시
- [ ] 전국 17개 시도 모두 지원
- [ ] 선택값은 로컬에 저장되어 다음 실행 시 유지

---

## F-10: 설정 화면 [P0]

### 설명
앱 전역 설정을 관리한다.

### 수용 기준
- [ ] 주 사용 유종 설정 (휘발유/경유/고급휘발유/등유)
- [ ] 즐겨찾기 가격 변동 알림 on/off
- [ ] 알림 기준 변동 폭 설정 (±10원/±20원/±50원)
- [ ] 검색 반경 설정 (1km/3km/5km/10km)
- [ ] 앱 버전, 오픈소스 라이선스, 개인정보처리방침
- [ ] **"데이터 새로고침"** 버튼 (수동 캐시 무효화)

## F-11: 온보딩 [P1]

### 설명
첫 실행 시 앱의 가치 제안을 3단계로 설명한다.

### 수용 기준
- [ ] 3개 화면 스와이프 (각각 1줄 메시지 + 일러스트)
  1. "유가 급등 시대, 우리 동네 최저가를 찾아드려요"
  2. "정부 최고가와 비교해 합리적인 주유"
  3. "주유 기록을 자동으로 관리"
- [ ] 마지막 화면에서 위치 권한 요청
- [ ] Skip 버튼으로 건너뛰기 가능
- [ ] 한 번 본 후에는 재노출 안 됨 (`shared_preferences`로 플래그 저장)

## F-12: 정부 최고가격제 배너 [P2]

### 설명
홈 화면 상단에 정부가 발표한 현재 최고가격을 배너로 표시한다.

### 수용 기준
- [ ] 배너 내용: "정부 최고가: 휘발유 1,934원 / 경유 1,923원 (3차, 2026.04.10 발표)"
- [ ] 탭 시 최고가격제 설명 모달 (간단한 FAQ)
- [ ] 데이터는 앱 내 하드코딩 (v1) → API 연동 (v2)

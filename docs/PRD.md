# PRD: FuelKeeper

> Product Requirements Document  
> Version 1.0 | 2026.04.22  
> Author: PM (Portfolio Project)

---

## 1. Executive Summary

**FuelKeeper**는 실시간 주유소 가격 비교 및 주유 가계부 기능을 제공하는 Flutter 모바일 애플리케이션이다. 한국석유공사 오피넷 공공 API를 활용해 사용자 현재 위치 기반 최저가 주유소를 추천하고, 정부 석유 최고가격제 대비 비교, 주유 기록 자동 관리, 홈 화면 위젯을 통한 원클릭 정보 접근을 제공한다.

---

## 2. 배경 및 문제 정의

### 2.1 시장 상황 (2026년 4월 기준)

- **국제유가 급등:** 원유가 배럴당 $90.45, 전년 대비 **+42%** 상승
- **국내 유가:** 전국 평균 휘발유 L당 1996.2원, **L당 2000원 돌파 임박**
- **지역 편차 심화:** 서울(2026원), 제주(2029원), 충북(2001원), 경기(2001원) 이미 2000원 초과
- **정부 개입:** 석유 최고가격제 3차 발표 — 휘발유 1934원, 경유 1923원
- **트리거 이벤트:** 미국의 이란 해상봉쇄 조치로 공급 불안 가중

### 2.2 사용자 Pain Point

| Pain Point | 근거 |
|---|---|
| **기존 공식 앱(오피넷)의 UX 악화** | 앱스토어 리뷰 다수 악평. "업데이트 후 주유소 이름이 짤려서 안 보임", "비교가 한눈에 안 됨", "데이터 최신화 안 됨" |
| **정부 최고가격 대비 정보 부재** | 현재 어떤 앱도 "이 주유소가 정부 최고가 대비 얼마나 싼지/비싼지"를 직관적으로 보여주지 않음 |
| **주유 기록 수작업** | 가계부 앱은 범용이라 주유 특화 기능(리터당 단가, 연비 계산) 부재 |
| **정보 접근성** | 운전 중에도 확인 가능한 홈 화면 위젯 부재 |

### 2.3 왜 지금인가 (Why Now)

- 유가 급등은 **전 국민 체감 이슈** — 실사용자 확보 용이
- 오피넷 공식 API가 **무료로 개방**되어 있어 진입 장벽 낮음
- 기존 앱의 명확한 취약점 → 차별화 포인트 뚜렷
- **포트폴리오 타이밍:** 시의성 있는 프로젝트는 면접 스토리텔링에 강력함

## 3. 제품 비전 및 목표

### 3.1 비전 (Vision)

> "운전자가 주유소에 도착하기 전에, **이미 결정이 끝나있는** 경험"

### 3.2 미션 (Mission)

복잡한 가격 비교를 3초 안에 끝낼 수 있도록, 필요한 정보만 정확하게 보여준다.

### 3.3 성공 지표 (Success Metrics)

**포트폴리오 관점:**
- 실제 배포된 APK + 웹 랜딩 페이지 존재
- 실사용자 최소 10명 이상 확보 (지인 테스트)
- 면접에서 기술 챌린지 3개 이상을 5분씩 설명 가능

**제품 관점 (가상 KPI):**
- 앱 첫 실행부터 최저가 주유소 확인까지 **5초 이내**
- 홈 위젯으로 앱 실행 없이도 가격 확인 가능
- 주요 기능 오프라인에서도 동작 (마지막 캐시 기반)

## 4. 타겟 사용자

### 4.1 페르소나

**Primary: 통근 운전자 (30~50대)**
- 주 2~3회 주유, 월 15~25만원 주유비 지출
- 500m~2km 내 단골 주유소가 있지만 최저가인지 확신 없음
- "어차피 그 동네인데 100원 차이가 큰가?" → 월 환산 시 수만원 차이를 인지하지 못함

**Secondary: 장거리 운전자 / 법인차 운전자**
- 경로 상 어디서 주유할지 매번 고민
- 세금 처리용 주유 기록 필요

**Tertiary: 절약 지향 20~30대**
- 첫 차 구매 후 유지비에 민감
- SNS 공유 욕구 ("우리 동네 최저가는 여기!")

### 4.2 타겟이 아닌 사용자

- 전기차 전용 사용자 (v1 범위 외, v2에서 충전소 추가 고려)
- 법인 대량 관리자 (B2B는 별도 제품)

## 5. 핵심 가치 제안 (Value Proposition)

| 경쟁 대비 차별점 | 설명 |
|---|---|
| **정부 최고가 비교** | 각 주유소 가격 옆에 "최고가 대비 -62원" 표시. 불법/비싼 주유소 즉시 식별 |
| **홈 화면 위젯** | 앱 실행 없이 우리 동네 최저가 3곳을 홈 화면에서 확인 |
| **주유 가계부 자동화** | 결제 후 "얼마 넣으셨나요?" 한 번만 입력하면 자동으로 리터/단가/연비 계산 |
| **가격 급등 알림** | 국제유가 기반 "내일 10원 오를 수 있어요" 푸시 |
| **운전자 중심 UX** | 큰 폰트, 색약 고려 컬러, 한 손 조작 가능한 바텀 UI |

## 6. 범위 (Scope)

### 6.1 v1.0 (3주 내 완성 - 포트폴리오 제출용)

✅ **포함:**
- 오피넷 API 연동 및 실시간 가격 조회
- 지도 기반 주변 주유소 탐색 (반경 3km)
- 리스트 뷰 (최저가 정렬, 브랜드 필터)
- 주유소 상세 정보 (가격, 셀프 여부, 세차장, 전화, 길찾기 연동)
- 관심 주유소 즐겨찾기
- 주유 기록 CRUD + 월별 통계
- 정부 최고가격 대비 비교
- 홈 화면 위젯 (Android 우선)
- 로컬 푸시 알림 (관심 주유소 가격 변동 시)

❌ **제외 (v2 이후):**
- 전기차 충전소
- LPG 충전소 (v1.1에서 추가 고려)
- 소셜 기능 (공유, 리뷰)
- 결제/페이백 연동
- iOS 위젯 (Apple Developer 미가입)

### 6.2 명시적 제약 사항

- **데이터 출처:** 오피넷 API의 데이터만 신뢰한다. 사용자 수동 입력 가격 수집은 하지 않는다 (데이터 오염 방지)
- **서버리스 지향:** v1은 백엔드 서버를 두지 않고, 공공 API + 로컬 DB만으로 구동한다 (운영 비용 0원, 개인정보 처리 리스크 제거)
- **위치 정보:** 서버로 전송하지 않고 기기 내에서만 사용한다

## 7. 사용자 스토리 (User Stories)

### Epic 1: 최저가 찾기
- **US-01:** 사용자로서, 앱을 열자마자 내 주변 최저가 주유소 3곳을 보고 싶다.
- **US-02:** 사용자로서, 특정 브랜드(SK, GS, 현대, S-Oil, 알뜰)만 필터링하고 싶다.
- **US-03:** 사용자로서, 최저가 주유소에 바로 전화를 걸거나 길찾기를 실행하고 싶다.

### Epic 2: 스마트한 비교
- **US-04:** 사용자로서, 이 주유소가 우리 동네 평균 대비 싼지 비싼지 한눈에 알고 싶다.
- **US-05:** 사용자로서, 정부 최고가격을 초과하는 주유소는 경고로 표시되길 원한다.

### Epic 3: 관심 주유소 관리
- **US-06:** 사용자로서, 자주 가는 주유소를 즐겨찾기에 등록하고 싶다.
- **US-07:** 사용자로서, 즐겨찾기 주유소의 가격이 변동되면 알림을 받고 싶다.

### Epic 4: 주유 기록
- **US-08:** 사용자로서, 주유 후 금액만 입력하면 자동으로 리터/단가를 기록하고 싶다.
- **US-09:** 사용자로서, 이번 달 주유비와 평균 단가를 차트로 보고 싶다.
- **US-10:** 사용자로서, 연비(km/L)를 자동 계산받고 싶다.

### Epic 5: 홈 위젯
- **US-11:** 사용자로서, 앱을 열지 않고도 홈 화면에서 최저가를 확인하고 싶다.
- **US-12:** 사용자로서, 위젯을 탭하면 해당 주유소로 바로 길찾기가 실행되길 원한다.

## 8. 비기능 요구사항 (Non-Functional Requirements)

| 항목 | 요구사항 |
|---|---|
| **성능** | 앱 진입 후 첫 화면 렌더링 1.5초 이내, 지도 로딩 3초 이내 |
| **오프라인** | 마지막 조회 데이터는 캐시로 보여주되, "30분 전 정보" 명시 |
| **접근성** | 최소 폰트 16sp, 컬러 대비 WCAG AA 이상 |
| **API 쿼터** | 오피넷 API 일일 한도 초과 방지를 위해 30분 TTL 캐싱 |
| **배터리** | 백그라운드 위치 추적 없음. 앱 실행 시에만 위치 조회 |
| **보안** | API 키는 flutter_dotenv + obfuscation으로 보호 |

## 9. 리스크 및 대응

| 리스크 | 영향도 | 대응 |
|---|---|---|
| 오피넷 API 쿼터 초과 | 높음 | 30분 TTL 캐싱, 에러 시 마지막 캐시 표시 |
| API 응답 지연/실패 | 중간 | 3초 타임아웃 + Fallback UI + 재시도 버튼 |
| 위치 권한 거부 | 중간 | 수동 지역 선택(시/군/구) 대체 플로우 제공 |
| 데이터 신선도 | 중간 | 화면에 "마지막 업데이트 시각" 명시 |
| iOS 미지원 | 낮음 | 포폴 단계에서는 Android APK만으로 충분 (README 명시) |
